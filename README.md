# NYT Crossword++

Improve your crossword solving experience on NYT Games! 
- Add a keyboard shortcut to toggle pencil on and off (defaults to ".")
- Customize the color of the pencil squares to make them easier to see
- Add a keyboard shortcut to enter rebus squares

Works on the Daily, Midi, and Mini puzzles.

**Privacy Policy:
**
This extension collects no user data and does not sell or share anything with 3rd parties.
